Mapa do site

 

Descrição das páginas

Página inicial

A página inicial contém uma lista das notícias mais recentes publicadas. O menu do site é apresentado na lateral esquerda do site. Nele é possível navegar de maneira rápida por todas as categorias existentes de notícias. Por meio dele, também é possível acessar as páginas: notícias da categoria, criar uma conta, recuperar a senha ou se autenticar se já possuir uma. Se houver usuário autenticado, na seção do usuário deste menu são apresentadas as opções disponíveis de acordo com o nível de acesso do usuário.

Notícias da categoria

A página notícias da categoria contém uma lista de notícias associadas àquela categoria e organizadas por ordem de publicação mais recente.

Leitura de notícia

A página leitura de notícia exibe a notícia que tiver sido selecionada, os comentários e quantas pessoas avaliaram como relevante a notícia. Se o usuário estiver autenticado também poderá registrar um comentário ou marcar a notícia como relevante.

Entrar

A página entrar contém um formulário que permita o usuário se autenticar no site informando seu e-mail e senha. Por meio dessa página também é possível acessar a página para recuperação de senha (Criar conta).

Criar conta

A página criar conta contém o formulário que deverá ser usado por todos os usuários do site para recuperar a senha ou criar uma conta. Apenas o usuário administrador é que terá uma conta já criada. O campo e-mail deve ser o primeiro a ser preenchido. O sistema deverá verificar se já existe algum usuário com o e-mail informado. Se existir, um link para redefinição de senha deverá ser enviado por e-mail ao usuário. Caso contrário, o restante do formulário será exibido para que o usuário possa preencher os demais campos.

Dados pessoais

A página dados pessoais contém os dados fornecidos pelo usuário no momento de criar a conta, bem como os meios necessários para mantê-los.

Lista de (categorias | notícias | usuários)

Um recurso pode ser uma categoria, uma notícia ou um usuário. Esta página contém uma lista com todos os registros criados do recurso selecionado, bem como os meios necessários para mantê-los. Caso a lista seja de recursos do tipo usuário haverá a opção de promovê-lo ou despromovê-lo.

Edição de (dados pessoais | categoria | notícia)

Esta página contém um formulário para editar cada um dos campos pertinentes ao recurso selecionado.
